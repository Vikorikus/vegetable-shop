Магазин овощей на Vite + React + Vitest и RTL

https://vikorikus.github.io/vegetable-shop/

Описание тестов
Unit-тесты: проверка корректности отображения данных в карточке товара и работы локальных счетчиков.

Интеграционные тесты: проверка связи между каталогом товаров и компонентом корзины в шапке сайта, включая имитацию работы API.
